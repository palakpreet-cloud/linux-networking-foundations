# Linux Networking Commands

Networking commands are essential for diagnosing connectivity and server issues.

---

## Check IP Address

Command:
ip a

Shows network interfaces and IP addresses.

---

## Test Connectivity

Command:
ping google.com

Checks if the system can reach an external server.

---

## View Open Ports

Command:
ss -tuln

Displays listening ports and active connections.

---

## Fetch Web Content

Command:
curl google.com

Sends HTTP request and retrieves response.

---

## DNS Lookup

Command:
nslookup google.com

Resolves domain name to IP address.
