# TCP/IP Model

The TCP/IP model is a simplified 4-layer networking model used in real-world internet communication.

---

## 4 Layers

1. Network Access Layer  
2. Internet Layer  
3. Transport Layer  
4. Application Layer  

---

## TCP vs UDP

| Feature | TCP | UDP |
|----------|------|------|
| Reliability | Reliable | Not reliable |
| Connection | Connection-oriented | Connectionless |
| Speed | Slower | Faster |
| Used In | HTTP, HTTPS | DNS, Streaming |

---

## Why TCP is Important in Cloud

- Ensures reliable communication
- Used in web servers
- Important for secure connections
