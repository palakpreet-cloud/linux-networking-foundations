# Ports and Protocols

Ports are logical endpoints used by applications to communicate over a network.

A protocol defines the rules for communication.

---

## Common Protocols

- HTTP → Web communication
- HTTPS → Secure web communication
- FTP → File transfer
- SSH → Secure remote login
- SMTP → Email sending
- DNS → Domain name resolution

---

## Common Port Numbers

| Port | Protocol | Purpose |
|------|----------|----------|
| 22   | SSH      | Secure remote login |
| 80   | HTTP     | Web traffic |
| 443  | HTTPS    | Secure web traffic |
| 21   | FTP      | File transfer |
| 25   | SMTP     | Email sending |
| 53   | DNS      | Domain resolution |

---

## Why Ports Matter in Cloud

- Security groups control port access
- Firewalls allow/block ports
- Servers listen on specific ports
- Web applications typically run on port 80 or 443
